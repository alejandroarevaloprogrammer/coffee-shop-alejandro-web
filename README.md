# Café Alejandro — v.1

Web multidioma para una cafetería ficticia/personal llamada **Café Alejandro**.

## Stack

- Astro
- TypeScript
- Tailwind CSS
- Español, Català, English

## Ejecutar

```bash
npm install
npm run dev
```

## Rutas

- `/es`
- `/ca`
- `/en`

## Nota

Esta es la versión `v.1`: estructura base, home multidioma, componentes iniciales y foto principal.
